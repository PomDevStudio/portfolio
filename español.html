Sobre Mi

Sobre Mi

Soy un Diseñador de Sitios Web.

Hago sitios web para ganarme la vida, puedo diseñar sitios web para su empresa o para su uso personal. aquí en mi portafolio podrás ver los diversos proyectos en los que he estado trabajando, para que puedas decidir si quieres un sitio web mío o no, principalmente ocupo frameworks, como Bootstrap, MDBootstrap y más, pero en esta portafolio, solo usé css y javascript sin ayuda de frameworks.

He estado trabajando en el desarrollo web front-end desde 2019 y he estado refinando mis técnicas desde ese momento. Tengo experiencia en la personalización de sitios web y todos sus detalles para asegurarme de que obtenga su sitio web perfecto.

Nuestros Servicios

Aqui encontrara los servicios que proveemos.

Diseño Web

El Diseño Web es nuestro servicio principal, Diseñaremos un sitio web para ti o para tu empresa.

Desarrollo Web

El desarrollo web es parte de nuestros servicios principal, ya que significa que damos funcionalidad a su sitio web, este se centra más en cómo resultará su experiencia con el sitio.

Sitios web moviles

Tenemos la habilidad de hacer sitios web que se enfocan en dispositivos moviles, con cambios especificos y con detalles especificos que hacen una experiencia más funcional para los dispositivos moviles.

Fotografía

Nos enfocamos en buscar imagenes para sus sitios web, dandole personalidad y un lado más descriptivo a tus sitios.

Contenido especifico para areas

Usaremos los detalles que usted nos de para editar partes especificas y brindarles areas para lo lo que usted quiera en su sitio web (esto puede ser un estandarte, cabeza de pagina, o una seccion sobre lo que usted quiera).

Edicion de Videos

Ademas de nuestros servicios web, tambien podemos editar videos y fotos para hacerlos que se vean como usted quiera.
